# Azure Resource Group Automation

A Bash script that automates the creation of Azure resource groups
across four environments: Dev, Test, UAT, and Production.

## The Problem
Administrators were manually creating resource groups every week — slow and error-prone.

## The Solution
Run one command and all four groups are created automatically.

## How to Use
bash
./create_rg.sh Project-A
